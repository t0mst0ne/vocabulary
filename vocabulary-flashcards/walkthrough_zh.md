# 單字卡擴充功能 - 使用指南

我已經為您建立了單字卡 Chrome 擴充功能。

## 功能特色
- **單字卡介面**：現代化的玻璃擬態設計 (Glassmorphism)。
- **完整的詞彙庫**：
    - **來源**：更新自「高中英文參考詞彙表(111學年度起適用)」，包含 **6124 個單字**。
    - **詳細分析**：每張卡片都包含最常見的意義、較少見的意義、詞族 (Word Family) 和相關片語。
    - **離線優先**：核心定義和例句是使用 NLTK WordNet 在本地生成的。
        - **詞族**：包含衍生詞形式。
        - **片語**：(離線模式下支援有限)。
    - **外部資源**：
        - **劍橋辭典 (Cambridge Dictionary)**：提供直接連結和嵌入式定義。
        - **韋氏辭典 (Merriam-Webster)**：提供直接連結和嵌入式定義。
            - **簡潔檢視**：移除了普通單字的干擾連結。
            - **保留慣用語**：保留了相關慣用語和片語的連結，方便點擊探索。
- **測驗模式 (Quiz Mode)**：
    - 切換到「測驗 (Quiz)」分頁來測試您的知識。
    - **格式**：選擇題 (為給定的單字選擇正確的 **英文定義**)。
    - **進階挑戰**：測驗會優先考 **較少見的意義**，以測試更深層的理解。
    - **參考語境**：提供一個遮蔽關鍵字的 **例句**，讓您了解該意義的語境。
    - **深入學習**：點擊 **問題單字** 本身，可立即切換到「學習 (Learn)」分頁，查看其完整分析和劍橋辭典條目。
    - **錯誤筆記本**：自動記錄答錯的題目。
    - **下載**：點擊「下載錯誤 (Download Mistakes)」將您的錯誤記錄儲存為 JSON 檔案以便複習。
- **導覽**：
    - **隨機順序**：每次打開擴充功能時，單字都會重新洗牌，讓您的學習保持新鮮感。
    - **控制**：使用 下一個 (Next) / 上一個 (Previous) 按鈕在洗牌後的列表中導覽。
- **互動**：點擊卡片即可翻轉。

## 如何在另一台電腦上安裝 (例如：給您的孩子)

要與您的孩子分享此擴充功能，請按照以下步驟操作：

1.  **傳輸檔案**：
    *   我已經為您建立了一個壓縮檔：`vocabulary-flashcards.zip`。
    *   請將此檔案傳送到您孩子的電腦 (透過電子郵件、USB、AirDrop 等)。

2.  **解壓縮**：
    *   在孩子的電腦上，解壓縮 `vocabulary-flashcards.zip` 檔案。您應該會看到一個名為 `vocabulary-flashcards` 的資料夾。

3.  **在 Chrome 中載入**：
    *   在孩子的電腦上打開 Google Chrome。
    *   前往 `chrome://extensions/` (在網址列輸入此網址)。
    *   開啟右上角的 **開發人員模式 (Developer mode)**。
    *   點擊左上角的 **載入未封裝項目 (Load unpacked)**。
    *   選擇您剛剛解壓縮的 `vocabulary-flashcards` 資料夾。

4.  **釘選擴充功能**：
    *   點擊 Chrome 工具列中的拼圖圖示。
    *   找到 "Vocabulary Flashcards" 並點擊圖釘圖示，將其固定在工具列上。

現在您的孩子可以使用單字卡、進行測驗並追蹤他們的進度了！

## 如何使用
1. 點擊 Chrome 工具列中的擴充功能圖示 (紫色的 "V")。
2. 彈出視窗將打開並顯示第一個單字。
3. **點擊卡片** 翻轉它並查看詳細分析。
4. 使用 **Next (下一個) / Previous (上一個)** 按鈕在列表中導覽。

## 建立的檔案
- `manifest.json`：擴充功能設定檔。
- `popup.html`, `style.css`, `popup.js`：使用者介面和邏輯。
- `data/words.json`：單字資料庫。
- `images/`：生成的圖示。
